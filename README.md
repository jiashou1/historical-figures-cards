# 🎲 历史人物小卡片

一个基于通义千问 AI 的历史人物知识卡片生成器，支持随机展示历史人物并生成详细的生平资料。

## 在线体验

👉 [GitHub Pages](https://jiashou1.github.io/historical-figures-cards/)

## 功能特性

- **随机人物展示**：从 2524 位历史人物中随机选择
- **AI 生成内容**：调用通义千问 qwen-max 模型，启用联网搜索
- **流式输出**：AI 内容实时流式展示，无需等待
- **置信度标注**：每条信息标注置信度等级（高/中/低）
- **出处标注**：每个事件标注史料出处
- **风险提示**：AI 生成内容的风险警告

## 技术方案

| 项目 | 内容 |
|------|------|
| AI 服务 | 通义千问 (DashScope) |
| 模型 | qwen-max |
| 联网搜索 | 启用（enable_search: true） |
| API Key 存储 | 浏览器 localStorage |
| CORS 代理 | corsproxy.io（线上环境） |

## AI 幻觉应对方案

| 措施 | 说明 |
|------|------|
| 联网搜索 | 启用 Qwen 联网功能，基于实时搜索结果生成 |
| Prompt 约束 | 要求 AI 标注不确定信息，禁止编造出处 |
| 置信度标注 | 每条信息标注置信度等级 |

## 置信度说明

| 等级 | 颜色 | 含义 |
|------|------|------|
| 高 | 🟢 绿色 | 史料明确记载 |
| 中 | 🟡 黄色 | 有记载但存在争议/不确定 |
| 低 | 🔴 红色 | 基于推断或传说，存疑 |

## 使用方法

1. 打开应用，点击右上角 ⚙️ 图标
2. 输入通义千问 API Key（[点击获取](https://dashscope.console.aliyun.com/apiKey)）
3. 点击「换一个」按钮生成随机历史人物卡片
4. 等待 AI 流式生成内容

## 本地开发

```bash
# 克隆仓库
git clone https://github.com/jiashou1/historical-figures-cards.git

# 进入目录
cd historical-figures-cards

# 启动本地服务器（任选一种）
python3 -m http.server 8080
# 或
npx serve .

# 访问 http://localhost:8080
```
